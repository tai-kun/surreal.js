add

  • add<T>(runner, options?): StatefulPromise<T>

  • Add a task to the stack.

    The added task runner will remove itself from the stack upon completion, regardless of whether it succeeds or fails.

    Therefore, to ensure we get the result of the task runner, we need to keep the return value of this .add() method.

    The .dispose() method only terminates pending task runners, and we won't be able to reliably retrieve the results of the task runner afterward.
