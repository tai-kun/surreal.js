Class TaskEmitter<T>

The created TaskEmitter instance must be disposed of by calling .dispose() at the end.

emit

  • emit<K>(event, ...args): undefined | StatefulPromise<unknown>[]

  • Emit an event.

    We can use Promise.all to wait for the event listeners triggered by this emit to complete.

    If the emitter is properly disposed of using .dispose(), there is no need to wait for these Promises, and there will be no resource leaks.

    Type Parameters

    The promises for the listeners, if any.

    Example
